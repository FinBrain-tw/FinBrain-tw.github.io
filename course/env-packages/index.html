安裝miniconda

  • 選擇版本安裝
  • 開啟anaconda prompt 輸入 conda install jupyterlab
  • 點選settings ➜ Advanced Settings Editor ➜ 選擇Notebook ➜ 複製Defaults ➜ 將lineNumbers改成true



.

建立environment

  • 開啟anaconda prompt
    conda create -n FinBrain python=3.7
    conda activate FinBrain
    pip install ipykernel==5.5.3
    python -m ipykernel install --user --name FinBrain

.

安裝套件

  • 顯示目前套件: pip list
  • 存檔至user: pip freeze > requirements.txt

  • 改寫requirements.txt
requests==2.22.0
beautifulsoup4==4.9.3
selenium==3.141.0
ipywidgets==7.4.2
tqdm==4.31.1
seaborn==0.11.1
plotly==5.1.0
pyecharts==1.7.1
yfinance==0.1.62
fugle_realtime==0.2.9
tensorflow==1.14.0
scikit-learn==0.24.2
lightgbm==3.1.1
xgboost==1.4.2
keras==2.2.4
  • 安裝套件 pip install -r requirements.txt
  • 安裝talib
    • 下載存至user
    • pip install TA_Lib-0.4.20-cp37-cp37m-win_amd64.whl

  • 更新套件
    pip install pandas==1.1.3
    pip install numpy==1.20.0
  • jupyterlab plotly問題
    (base) jupyter labextension install jupyterlab-plotly@4.14.3
    重啟jupyter lab

.

下載中文字體

  • TaipeiSansTCBeta-Regular.ttf

  • 字體存至matplotlib\mpl-data\fonts\ttf
  • Users\apple.matplotlib刪除快取檔案fontList.json
  • 使用時輸入 plt.rcParams['font.sans-serif'] = ['Taipei Sans TC Beta']
