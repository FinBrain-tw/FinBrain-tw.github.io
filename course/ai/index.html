人工智慧

介紹

其實早在1950年代就已經有AI的概念，但為什麼直到2015年後開始爆炸性成長呢? 主要是因為Big Data的發展，隨著智慧型手機的誕生，電腦、手機、容量就必須越做越大，造成數據爆炸性地成長;再加上computational power(運算能力)等晶片、顯卡大幅提升造成ML、DL可以迅速發展。
應用 實例
憑感覺的問題-影像辨識 數字辨識
傳統方式難以最佳解 圍棋
資料量極大的問題 google翻譯

.

  • 機器學習 (Machine Learning)

定義
parse data, learn from data, make informed decisions, by using algorithms and statistical models to analyse and draw inferences from patterns in data.
訓練關鍵 意涵
Features 決定讓model從哪種角度去分析學習 / 需要domain knowledge
Label 正確解答
分類 Supervised (監督式學習) Unsupervised (非監督式學習) Reinforcements (Semi-Supervised) (強化學習)
特色 有features和label 只有features沒有label Agent(AI)與environment互動
原理 寫題目後對答案來修正 做相似度分析 (單純分類) 獎懲機制來訓練AI
優點 缺點
訓練的時間較短 預測的時間較久
模型種類
Support Vector Machine (SVM)
Random Forest (RF)
LightGBM (lgb)
XGBoost (xgb)

.

  • 深度學習 (Deep Learning) (Neural Network)

定義
artificial neural network, mimic human brain to learn
訓練關鍵
模仿人腦的神經元
不需要人為設計Features

優點 缺點
預測的時間較快 訓練的時間較久
模型種類
Keras model
Convolutional Neural Network (CNN)
Long short-term memory (LSTM)

.

結論

訓練風險
資料量不足
過多無用資料
過擬合 (overfitting)
未擬合 (underfitting)
其實機器學習就是透過各種model進行數學統計的運算並進行優化。
缺點是訓練和判斷的過程過度黑盒化，導致我們很難完全信任AI的判斷結果。然而機器學習的優勢也非常明顯，它可以觀察到數據中容易被忽略或難以觀察到的點。
在這大數據的時代統計學家都一致認為機器學習和神經網路絕對是未來重大的趨勢，甚至在金融界也已經逐漸發展出非常優秀的AI選股策略囉!
