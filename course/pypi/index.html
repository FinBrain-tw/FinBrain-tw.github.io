專案架構

Finbrain ------ FinBrain ----------------- __init__.py 
            |                        |
            |-- requirements.txt     |---- crawler.py
            |                        |
            |-- setup.py             |---- data.py
            |                        
            |-- .gitignore           
            |
            |-- README.md
關鍵架構
repo名稱(FinBrain) = 套件名稱(FinBrain)
套件需有__init__.py才會被打包上傳
requirements.txt 為專案所需下載的套件
setup.py的功能為上傳repo到PYPI
.gitignore為忽略不上傳到GitHub的檔案

.

建立專案

  • 安裝GitHub Desktop

  • 新增repo

clone完畢後的檔案位於C:\Users\apple\Documents\GitHub

.

  • 新增setup.py

setup範例檔
URL為repo的GitHub網址
REQUIRED為專案所需套件，可直接讀取requirements.txt

  • 新增.gitignore

gitignore範例檔

.

上傳GitHub

  • 專案移至文件/GitHub/MyProject

  • GitHub Desktop上傳

.

上傳PYPI

  • 申請API Token

  • (user)建立.pypirc

  • cmd專案資料夾

  • 安裝上傳套件

pip install twine
pip install wheel

  • 開始上傳

python setup.py sdist
python setup.py bdist_wheel
twine upload dist/*
上傳成功!

更新版本

  • setup.py修改版本

  • 開始上傳

python setup.py sdist
python setup.py bdist_wheel
twine upload dist/*
更新成功!

PYPI 和 GitHub 為不同專案的資料庫，所以即使下載同一個套件也有可能安裝到不同版本的packages
  • PYPI 安裝 > pip install MyProject
  • GitHub 安裝 > pip install git+https://github.com/User/MyProject.git
