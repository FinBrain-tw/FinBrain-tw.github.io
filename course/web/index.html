如何使用GitHub架網站

  • DURATION

    06 Month

  • WEEKLY

    03 hours

  • FEE

    From: $800

Apply now

建立網頁

  • 選擇網頁版面

  • exampleSite內新增themes資料夾，刪除資料夾名稱的-master放入themes資料夾

  • 安裝Hugo.exe，移至exampleSite

  • 運行網頁

cmd exampleSite ➜ 輸入hugo server ➜ 前往localhost

.

網頁設計架構

exampleSite ----- content ----- blog # 文章內容 md
               |
               |- data ----- homepage.yml # 首頁設計 yml
               |
               |- resources ----- assets # 版面配置 scss
               |
               |- static ----- images # 圖片
               |
               |- themes # 主題高階設定
               |
               |- config.toml # 個人主題

.

網頁上線

  • 設定個人網址

網址必須為個人帳號名稱

  • 建立專案

專案名稱必須為個人帳號名稱+github.io

  • cmd exampleSite ➜ 輸入hugo ➜ 複製public內的資料到文件/GitHub/UserName.github.io ➜ GitHub Desktop上傳

上傳成功!

.

添加google功能

  • Google Analytics

建立新帳戶

只建立通用 Analytics (分析) 資源

複製追蹤ID (UA-)

修改config.toml的google_analitycs_id

  • Google Search Console

選擇網址前置字元

  • 文章簡介設定

index.html:
<p class='card-text'>{{.Summary}} </p>中 .Summary➜ .Description

  • 字型設定

_main.scss:

/* markdown */
@import url('https://fonts.googleapis.com/css2?family=Noto+Sans+TC:wght@300&display=swap');
.content {
    *{
        font-family:'Noto Sans TC';
    }
    }
